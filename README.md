# How to

```zsh
git clone https://github.com/just-do-halee/myshell

chmod +x myshell/run.sh
```

And

<br>

##### If you are zsh user.

```zsh
. myshell/run.sh ~/.zshrc # executes
```

##### If you are bash user.

```bash
. myshell/run.sh ~/.bashrc # executes
```

##### If you are etc user.

```bash
. myshell/run.sh ~/.anyrc # executes
```
